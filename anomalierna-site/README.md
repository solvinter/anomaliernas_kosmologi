# anomalierna.solvinter.io

Minimal static website for Anomalierna.

The website is a rendered view of an underlying YAML archive. It is not intended
as a marketing page or as publication of finished output, but as a simple entry
point into the project's structure, method, and source material.

## Structure

```text
anomalierna-site/
├── index.html
├── style.css
├── yaml/
│   ├── methodology.yaml
│   ├── anomalies/
│   ├── categories/
│   └── relations/
└── output/
    ├── web/
    ├── reports/
    └── books/
```

Open `index.html` directly in a browser for local review.

## Public URL

The intended public host is `anomalierna.solvinter.io`.
